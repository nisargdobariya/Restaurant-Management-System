# 🍽️ Restaurant Management System

A comprehensive **Restaurant Management System** developed using **PHP** and **MySQL** to simplify restaurant operations such as menu management, order processing, billing, and administration. The system provides separate modules for administrators and staff, enabling efficient restaurant management through an easy-to-use web interface.

---

## 📖 Overview

Managing restaurant operations manually can be time-consuming and prone to errors. This project digitizes the complete workflow by allowing restaurant staff to manage customers, food items, orders, bills, and reports from a centralized system.

The application is built using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## ⚙️ Installation Guide

### 1. Clone the Repository

```bash
git clone https://github.com/nisargdobariya/Restaurant-Management-System.git
```

### 2. Move the Project

Copy the project folder into your web server directory.

For XAMPP:

```
C:\xampp\htdocs\
```

For WAMP:

```
C:\wamp64\www\
```

---

### 3. Start Services

Start:

- Apache
- MySQL

using XAMPP or WAMP Control Panel.

---

### 4. Import Database

1. Open phpMyAdmin
2. Create a new database (e.g., `restaurant`)
3. Import the SQL file located in:

```
DB/restaurant.sql
```

---

### 5. Configure Database Connection

Update your database connection file if necessary.

Example:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "restaurant";
```

---

### 6. Run the Project

Open your browser and visit:

```
http://localhost/project/
```

or

```
http://localhost/Restaurant-Management-System/
```

depending on your folder name.
